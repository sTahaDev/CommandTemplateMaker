# CommandTemplateMaker(CTM)
Basic Template Maker. It's for now only c++.

## How To Use
Use with Terminal <br>
```
Cmt [mode] [type]
```

## Commands
| mode | type |
| :---------: | :----------:|
|  new   |  -make  | 
| help  | -cmake  | 

## Example Usage
```
Ctm new -make
```

## Installation
Clone Repo
```
git clone https://github.com/sTahaDev/CommandTemplateMaker.git
```
Go Main Folder
```
cd CommandTemplateMaker
```
Go Bin Folder
```
cd bin
```
Run Ctm
```
./Ctm [mode] [type]
```
